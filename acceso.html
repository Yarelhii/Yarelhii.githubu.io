ACCESO BÁSICO A LA BASE DE DATOS

Septiembre, 2022

Cuando tengamos nuestro objeto llamdo CONNECTION, podremos empezar a usarlo para poder ejecutar nuestros comandos SQL en la Base de Datos .

Disponemos de tres tipos de sentencias en SQL

  • STATEMENT:

    • Representa una sentencia básica, sea de consulta(SELECT) o para poder manipular los datos(INSERT, UPDATE)

  • PREPAREDSTATEMENT:

    • Representa una sentencia precompilada de SQL que ofrece prestaciones

  • CALLABLESTATEMENT:

    • Representa una llamada a un procedimento para el almacenado en SQL
